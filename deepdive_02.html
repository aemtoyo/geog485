Question 2

Answer the following questions about a WMS GetCapabilities request

  1. What are the required parameters, and what do they represent?

    2. ANSWER:

    • WMTVER=1.0.0; Is asking is asking to request which version of the Web Mapping Service
    • SERVICE=WMS; Is asking what is the service type.
    • REQUEST=capabilities; Is asking for the name of service.
    • REQUEST=GetCapabilities; Is asking to get capabilities from the specified name.

  2. What is returned in response to a WMS GetCapabilities request?

    3. ANSWER: You should recieve an XML document.

Question 3

Answer the following questions about a WMS GetMap request

  1. What are the required parameters, and what do they represent?

    2. ANSWER:

    • WMTVER=1.0.0; Request version
    • VERSION=version; Request version.
    • REQUEST=map; Request name.
    • REQUEST=GetMap; Request name
    • LAYERS=layer_list; Comma-separated list of one or more map layers. Optional (ver. 1.1, 1.1.1) if SLD parameter is present.
    • STYLES=style_list; Comma-separated list of one rendering style per requested layer. Optional if SLD parameter is present.
    • SRS=namespace:identifier; Spatial Reference System.
    • CRS=namespace:identifier; Spatial Reference System.
    • BBOX=minx,miny,maxx,maxy; Bounding box corners (lower left, upper right)in SRS units.
    • WIDTH=output_width; Width in pixels of map picture.
    • HEIGHT=output_height; Height in pixels of map picture.
    • FORMAT=output_format; Output format of map.

  2. What is returned in response to a WMS GetMap request?

    3. ANSWER: The GetMap operation is designed to produce a map, which is defined to be either a pictorial image or a set of graphical elements.

  3. What is the significance of transparency in WMS requests?

    4. ANSWER: It allows you to put many layers on a single map.

Question 5

Which OGC request would you submit under the following circumstances (include both the service type [e.g. WMS, WFS, WCS], and the request [e.g. GetMap, GetCapabilities, GetCoverage, etc.] in your answer)

  1. You want a map image representing three layers of data in a single JPEG for a specified area of interest.

    2. ANSWER: WMS GetMap request

  2. You want to retrieve data representing geometries and associated attributes for a road network, with the returned data in GML.

    3. ANSWER: WFS GetFeature request

  3. You want to retrieve data representing a digital elevation model (a raster dataset) in the form of a GeoTIFF.

    4. ANSWER: WCS GetCoverage request
